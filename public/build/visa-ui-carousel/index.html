JS Interaction

  • onLoad:
    • If first card is now current card:
      • Add disabled="disabled" on .visa-ui-carousel-control--left
    • Else if last last is now current card:
      • Add disabled="disabled" on .visa-ui-carousel-control--right
    • Else (current card is neither first or last):
      • Remove disabled="disabled" from .visa-ui-carousel-control--left
      • Remove disabled="disabled" from .visa-ui-carousel-control--right
    • onClick:
      onKeyUp (Space bar):
      onKeyUp (Enter):
      • Left Arrow:
        • Remove .visa-ui-state--hide from previous card
        • Add .visa-ui-carousel-card--animation--left-to-default to previous card
        • Add .visa-ui-carousel-card--animation--default-to-right to current card
        • Wait until animation is done (250ms)
        • Make current card next card
        • Make previous card current card
        • Remove .visa-ui-carousel-card--animation--left-to-default from current card
        • Remove .visa-ui-carousel-card--animation--default-to-right from next card
        • Set every card after current to .visa-ui-carousel-card--right and .visa-ui-state--hide
        • Set every card before current to .visa-ui-carousel-card-left and .visa-ui-state--hide
        • Set current card to .visa-ui-carouself-card
        • Perform onLoad check
      • Right Arrow:
        • Remove .visa-ui-state--hide from next card
        • Add .visa-ui-carousel-card--animation--right-to-default to next card
        • Add .visa-ui-carousel-card--animation--default-to-left to current card
        • Wait until animation is done (250ms)
        • Make current card previous card
        • Make next card current card
        • Remove .visa-ui-carousel-card--animation--right-to-default from current card
        • Remove .visa-ui-carousel-card--animation--default-to-left from previous card
        • Set every card after current to .visa-ui-carousel-card--right and .visa-ui-state--hide
        • Set every card before current to .visa-ui-carousel-card-left and .visa-ui-state--hide
        • Set current card to .visa-ui-carouself-card
        • Perform onLoad check
